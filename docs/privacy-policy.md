# Privacy Policy

Last updated: March 16, 2026

Better GitHub PR View does not collect, sell, or transmit personal data to the developer or any third party.

## Data usage

- The extension reads GitHub pull request list and pull request detail pages on `github.com` so it can render additional metadata inside repository pull request lists.
- The extension stores user settings in browser sync storage.
- The extension stores a short-lived cache of fetched pull request metadata in browser local storage to reduce repeat page requests.

## Data sharing

- No analytics, telemetry, advertising, tracking pixels, or remote logging are included.
- No data is sent to the developer, a backend service, or external analytics tooling.
- Network requests stay within `github.com` and are used only to read page data needed to enrich the current GitHub UI.

## User controls

- Users can disable the extension entirely from the popup.
- Users can clear cached metadata from the popup at any time.
- Users can uninstall the extension to remove its stored settings and cache according to browser behavior.
