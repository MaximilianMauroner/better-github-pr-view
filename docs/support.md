# Support

Support for Better GitHub PR View is handled through the public repository.

## Get help

- Open an issue: `https://github.com/MaximilianMauroner/better-github-pr-view/issues`
- Review the README for build, packaging, and manual install steps
- Include browser version, extension version, affected GitHub URL pattern, and a screenshot when reporting a bug

## Store submission note

Chrome Web Store and Firefox Add-ons both ask for a support contact. The public support URL above is ready to use. If a support email address is required in a dashboard field, add the maintainer-managed address there during submission.
